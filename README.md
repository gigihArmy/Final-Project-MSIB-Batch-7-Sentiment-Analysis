# Built With
- Streamlit Framework
- Gemini API
# Dataset
Media Social Dataset from Celerates (PT Mitra Talenta Grup)
The Dataset contains 500 rows from 3 Platfoms (Facebook, Twitter, and Instagram
# Features
- Exploratory Data Analysis on Media Social Dataset
- Text Pre-processing
- Feature Extraction with TF-IDF
- SVM Algorithm for Sentiment Classification into 3 classes (positive, neutral, and negative)
- Model Evaluation (Confusion Matrix)
- Gain insights on prediction result by utilizing Gemini Flash
